# Florence (피렌체) - Product Specification

## 0. Project Overview
- **앱 이름**: 피렌체 (Florence)
- **핵심 의의**: AI 독서 파트너, 함께하는 프로젝트 독서, 기존 독서 앱의 사용자 불편함 개선.

## 1. 초기 온보딩 (First Impression)
- **비주얼 가이드**: 앱 설치 후 첫 실행 시, 유저의 '내 서재'와 '함께한 독서' 데이터가 쌓였을 때의 완성된 모습을 미리 시각적으로 보여주어 기대감을 형성한다.
- **가치 제안**: 이 앱을 쓰면 어떤 것을 할 수 있는지 명확히 제시한다.

## 2. 주요 화면 구성 및 기능

### ① 홈 (Home)
- **상단 탭**: [내 서재] / [함께한 독서] 선택 가능 (앱 실행 시 '내 서재'가 기본 노출).
- **중앙 영역**: 선택한 탭에 따른 데이터 리스트 노출.
- **내 서재 탭**: 최상단에 '책 검색하기' 문구가 적힌 검색창 UI 상시 배치. 하단에는 기록해둔 책들 노출.
- **함께한 독서 탭**: 클릭 시 관련 프로젝트 페이지 정보 노출.
- **하단 네비게이션 바**: [홈] - [메모] - [읽는 중/읽고 싶은 책] - [마이페이지]

### ② 내 서재 (Main Storage)
- **시각화 (Physicality)**: '북적북적' 스타일의 책 쌓기 형식 차용. 단, 뉴모피즘을 적용해 입체감을 주고 페이지 수에 따라 다른 두께를 적용한다.
- **기록 방식**: 
  - 검색 후 '읽은 책' 선정 시 즉시 서재에 추가.
  - '읽고 싶은 책' 선정 시 [읽는 중/읽고 싶은 책] 보관함으로 이동하며, 이후 상태 변경 시 서재에 반영.
- **특장점**: 읽다 만 책, N회독 한 책 등 기존 앱에서 놓치기 쉬운 상태까지 섬세하게 기록 지원.

### ③ 메모 (Memo)
- **진입 경로**: [읽는 중/읽고 싶은 책] 탭 내 도서 표지 옆 '메모 작성하기' 버튼.
- **주요 기능**: 사진 내 텍스트 추출(OCR), 이미지 첨부, 직접 텍스트 입력 가능.
- **UI (북모리 스타일)**: [메모] 탭 클릭 시 책 표지들이 먼저 그리드로 뜨고, 특정 표지 선택 시 그동안 기록한 문답과 내용이 타임라인/리스트 형태로 나열됨.

### ④ 함께한 독서 (Project & Social)
- **소셜 UI**: 상단에 '북 메이트' 프로필 (인스타그램 스토리 형식). 동그란 이미지 밑에 닉네임이 작게 뜨며, 이미지 클릭 시 해당 유저의 '서재 구경하기'로 이동.
- **하단 리스트**: 그동안 프로젝트 성공으로 받은 영수증들과 함께한 멤버들 노출.
- **프로젝트 시스템**:
  - **시작하기 버튼**: 북 메이트 프로필과 기록 사이에 위치. (도서 선정 기능 포함)
  - **완수 기준**: 전원 완독. (실시간 진도 및 질문 개수/내용 팀원 간 공유 및 알림 기능)
  - **아카이빙**: 외부 AI(ChatGPT, Gemini 등)와 나눈 가치 있는 문답을 복사해 앱 내에 체계적으로 저장.

### ⑤ 마이페이지 (My Page)
- **사용자 정보**: 로그인 및 기타 계정 정보 관리.
- **독서 통계**: 연도별/월별 읽은 권수 통계 데이터 제공.
- **장르 분석**: 읽은 책의 장르 분포도 시각화 (예: 심리학 70%, 경제 22% 등).

## 3. 핵심 리워드 및 확산 (Output & Viral)

### 🧾 독서 영수증 (Result Paper)
- **발급 조건 및 동적 렌더링**: 
  - **100% 완수**: 팀원 모두에게 정상적이고 깔끔한 영수증 발급.
  - **50% 이상 ~ 100% 미만**: 한 달 경과 시, 구겨진 UI의 영수증 발급.
  - **50% 미만**: 발급 실패 (발급 안 됨).
- **디자인**: 힙한 감성의 디자인 (표지 이미지, 참여자, 기간 포함).
- **개별화 요소**: 공통 정보 외에 본인이 나눈 **'핵심 Q&A'**를 개별화하여 영수증에 반영.
- **연동**: 완료 시 '내 서재'에 해당 책이 자동으로 쌓임 + 함께한 독서 탭에 영수증 이미지 추가.



## 4. 디자인 시스템 (Design System)
- **폰트 (Typography)**: **Pretendard**를 기본 폰트로 사용하여 가독성과 모던함을 확보.
- **컬러 팔레트 (Color Palette)**:
  - **Primary**: **Dark Wine (#751013)** - 깊고 진한 와인색. 브랜드 메인 컬러.
  - **Background**: **Bright Snow (#F6F6F6)** - 깨끗하고 밝은 눈색 배경.
  - **Text**: **Graphite (#2E2E2E)** - 진한 흑연색 텍스트.
- **디자인 컨셉**: **Modern Minimal Luxury** (심플 & 트렌디)
  - **Flat & Clean**: 뉴모피즘의 복잡한 그림자를 제거하고, 간결한 라인과 면을 강조.
  - **Whitespace**: 충분한 여백을 두어 고급스러운 분위기 연출.
  - **Typography Oriented**: 컨텐츠와 텍스트 위주의 트렌디한 레이아웃.
  - **Soft Elevation**: 아주 옅은 그림자나 얇은 테두리(Border)로 세련된 구분감 제공.
